# Tetra

<img src="public/cover.jpg" />

Style guide and UI components for Chromatic's docs and marketing sites. https://tetra.chromatic.com

## Install

```console
pnpm add @chromatic-com/tetra
```

```console
pnpm run storybook
```

## Building

Build package with `tsup` for production.

```console
pnpm run build
```
