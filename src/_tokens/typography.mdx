# Typography

Like in other ui in mondays interface, typography works by principle of accessibility before aesthetics. Therefore the text should be readable and help the user understand what's important by well contrasted size and colors hierarchy.

<InfoBox backgroundColor="pink100">
  These typography tokens are not meant to be used on their own. At all cost you
  should avoid to use them directly in your code. Instead, use the typography
  components provided by the UI kit.
</InfoBox>

### Font Family

- `sans` "Nunito Sans", "Helvetica Neue", Helvetica, Arial, sans-serif
- `mono` "SFMono-Regular", Consolas, "Liberation Mono", Menlo, Courier, monospace

### Font Weight

- `regular` 400
- `medium` 500
- `semibold` 600
- `bold` 700
